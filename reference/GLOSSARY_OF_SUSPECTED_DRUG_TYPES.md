# Categories of drug for offenses that are cataloged in the NIBRS

Categories of drug for offenses that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_SUSPECTED_DRUG_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 18 rows and 2
columns

- suspected_drug_code:

  `<chr>` a two-digit identifier for the type of drug

- suspected_drug_desc:

  `<chr>` a human-readable description for the type of drug
