# Categories of property damage for offenses that are cataloged in the NIBRS

Categories of property damage for offenses that are cataloged in the
NIBRS

## Usage

``` r
GLOSSARY_OF_PROP_LOSS_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 8 rows and 2
columns

- prop_loss_code:

  `<chr>` a two-digit identifier for the type of damage

- prop_loss_desc:

  `<chr>` a human-readable description for the type of damage
