# Categories of law enforcement assignment that are cataloged in the NIBRS

Categories of law enforcement assignment that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_ASSIGNMENT_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 7 rows and 2
columns

- assignment_type_code:

  `<chr>` a two-digit identifier for the assignment type

- assignment_type_name:

  `<chr>` a human-readable description of the assignment type
