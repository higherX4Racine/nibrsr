# Categories of victim for offenses that are cataloged in the NIBRS

Categories of victim for offenses that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_VICTIM_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 9 rows and 2
columns

- victim_code:

  `<chr>` a two-digit identifier for the type of victim

- victim_desc:

  `<chr>` a human-readable description for the type of victim
