# Categories of law enforcement activities that are cataloged in the NIBRS

Categories of law enforcement activities that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_ACTIVITY_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 11 rows and 2
columns

- activity_type_code:

  `<chr>` a two-digit identifier for the activity type

- activity_type_name:

  `<chr>` a human-readable description of the activity type
