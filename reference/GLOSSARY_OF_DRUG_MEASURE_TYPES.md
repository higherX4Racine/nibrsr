# Units of drug quantities that are cataloged in the NIBRS

Units of drug quantities that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_DRUG_MEASURE_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 11 rows and 2
columns

- drug_measure_type_code:

  `<chr>` a two-digit identifier for the unit type

- drug_measure_type_name:

  `<chr>` a human-readable description of the unit type
