# Categories of venues for crimes that are cataloged in the NIBRS

Categories of venues for crimes that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_LOCATION_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 47 rows and 2
columns

- location_type_code:

  `<chr>` a two-digit identifier for the venue

- location_type_name:

  `<chr>` a human-readable description of the venue
