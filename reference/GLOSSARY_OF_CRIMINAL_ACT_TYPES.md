# Categories of criminal acts that are cataloged in the NIBRS

Categories of criminal acts that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_CRIMINAL_ACT_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 17 rows and 3
columns

- criminal_act_code:

  `<chr>` a two-digit identifier for the type of criminal act

- criminal_act_name:

  `<chr>` a short label for the type of criminal act

- criminal_act_desc:

  `<chr>` a human-readable description for the type of criminal act
