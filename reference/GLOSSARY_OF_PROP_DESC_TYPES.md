# Categories of property damage for offenses that are cataloged in the NIBRS

Categories of property damage for offenses that are cataloged in the
NIBRS

## Usage

``` r
GLOSSARY_OF_PROP_DESC_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 68 rows and 2
columns

- prop_desc_name:

  `<chr>` a human-readable description for the type of property

- prop_desc_code:

  `<chr>` a two-digit identifier for the type of property
