# Categories of arrest that are cataloged in the NIBRS

Categories of arrest that are cataloged in the NIBRS

## Usage

``` r
GLOSSARY_OF_ARREST_TYPES
```

## Format

an object of type `spec_tbl_df/tbl_df/tbl/data.frame` with 3 rows and 2
columns

- arrest_type_code:

  `<chr>` a two-digit identifier for the arrest type

- arrest_type_name:

  `<chr>` a human-readable description of the arrest type
